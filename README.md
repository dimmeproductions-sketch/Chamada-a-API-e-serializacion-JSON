# Chamada-a-API-e-serializacion-JSON
Neste exercicio utilizando UnityWebRequest, JSONUtility e corutinas deberás chamar á API:  https://opentdb.com/
Solicitando 10 preguntas. Esta API devolve un JSON que deberás serializar, creando as clases que coincidan coa estructura do JSON.
Crea a variable cos datos como un atributo público do script principal. Imprime na consola o número de preguntas solicitadas e 4 datos da primeira pregunta.
